# Typora Academic CV 📄

这是一个专为求职和学术场景打造的极简 Typora Markdown 简历模板。告别传统富文本编辑器里繁琐的表格调整与排版折腾，只需专注简历内容本身。通过简单的 Markdown 语法和本仓库提供的专属 CSS 样式，即可一键渲染并导出排版干净、专业的 PDF 简历。

## 📦 仓库包含内容

本仓库提供中英文两套模板，并各自匹配了深度优化的视觉与排版样式：

### 1. 中文简历套件 (学术/量化/常规求职)
*   **`简历.md`**：结构包含教育背景、科研成果（发表论文与专利）、项目经历等核心模块，非常适合需要展示详实经历的求职者或硕博科研人员。
*   **`academic-resume-zh.css`**：中文专属样式。内置中文字体优化（宋体/黑体与英文字体优雅混排），并以深蓝色（`#102a43`）作为重点强调色，视觉层次现代、清晰且专业。
*   **预览效果**：参考同目录下的 `简历.pdf`。

### 2. 英文简历套件 (外企/通用求职)
*   **`cv.md`**：标准的英文求职模板，包含 Summary, Education, Professional Experience, Leadership & Activities 等模块。
*   **`academic-resume-en.css`**：英文经典样式。采用纯正的 Times New Roman 衬线字体族，纯黑白配色（`#000000`），完美契合海外求职及严肃学术场合对排版的极致规范要求。
*   **预览效果**：参考同目录下的 `cv.pdf`。

## 🚀 使用指南

1.  **下载仓库**：将本项目克隆或下载 ZIP 到本地。
2.  **安装 CSS 主题**：
    *   打开 Typora。
    *   进入 `偏好设置 (Preferences) -> 外观 (Appearance)`。
    *   点击 `打开主题文件夹 (Open Theme Folder)`。
    *   将本仓库中的 `academic-resume-zh.css` 和 `academic-resume-en.css` 复制进该文件夹。
3.  **应用主题**：重启 Typora，在顶部菜单栏的 `主题 (Theme)` 中，根据你当前编辑的简历语言选择对应的样式名称。
4.  **撰写简历**：用 Typora 打开对应的 `.md` 模板文件，替换为你自己的个人信息。
5.  **导出 PDF**：排版满意后，点击 `文件 (File) -> 导出 (Export) -> PDF`，即可获得完美的 PDF 简历。

## 🛠️ 高级自定义

如果你想修改简历的主题颜色，可以用任意文本编辑器打开对应的 `.css` 文件，找到最顶部的 `:root` 变量区直接修改色值（以中文样式为例）：

```css
:root {
  --resume-text: #1f2933;   /* 主文本颜色 */
  --resume-muted: #536171;  /* 次要信息颜色 */
  --resume-accent: #102a43; /* 强调色（如大标题、重点文字） */
  --resume-rule: #c3ccd5;   /* 分割线颜色 */
}
```
